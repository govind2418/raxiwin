Raxi App

Raxi App — install Raxi Game on your home screen

The Raxi App is a lightweight installable version of Raxi Game — a couple hundred kilobytes, no app store review, no update downloads. Once installed, it opens full-screen like a native app and keeps working even with a spotty connection.

Play in Browser Instead

How to install the Raxi App

Android / Desktop Chrome

One tap install

Tap "Install Raxi App" above, or open the browser menu and choose Install app / Add to Home screen.

iPhone / iPad

Add to Home Screen

Open rexigame.com in Safari, tap the Share icon, then choose Add to Home Screen.

Any device

No install needed

Prefer not to install anything? Every game runs the same way straight from the Games hub in your browser.

What makes the Raxi App different

It's not a native download from an app store — it's a Progressive Web App (PWA), which means it installs in seconds, updates instantly when we ship changes, and never asks for storage permissions or account access it doesn't need.

The Raxi App is completely free with no in-app purchases, ads for real-money betting, or wagering features of any kind.

Free browser games with a dark, competitive edge. No downloads, no real-money wagering — just fast, skill-based play.
